# BitLocke驱动器加密
如果使用的是 Windows 11 专业版、企业版系统，可以通过 BitLocker 对整个磁盘分区加密：

>步骤：
>在任务栏搜索 “BitLocker”，打开 “BitLocker 驱动器加密”。
>找到你存放敏感文件夹的分区，点击“启用 BitLocker”。设置密码，加密整个分区。
>每次重启或插入移动设备（如 U 盘）时会要求输入密码。

>📌 缺点：不是按文件夹加密，而是整个磁盘。
