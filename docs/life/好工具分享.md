# 好工具分享

## 一、文件传输

LocalSend开源地址：https://github.com/localsend/localsend/releases/tag/v1.16.1

操作流程：手机与电脑端安装客户端，连接同一WiFi，一端选择文件，搜索设备发送，另一端点击确定，即可完成传输（支持所有客户端mac，）
