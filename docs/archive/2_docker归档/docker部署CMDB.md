# docker部署CMDB

> 项目地址：https://github.com/veops/cmdb?tab=readme-ov-file

# 一、Docker 一键快速构建

```
1.安装 Docker 环境和 Docker Compose（v2）
2.克隆项目代码, `git clone https://github.com/veops/cmdb.git`
3.进入主目录并启动, `docker compose up -d`
```
