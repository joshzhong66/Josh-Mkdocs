# 查看引导固件相关命令



```bash
# 检查系统是否以 EFI 模式启动，如果系统以传统 BIOS 模式启动，不会输出任何内容
ls /sys/firmware/ | grep efi

# 显示内核环形缓冲区中的日志，搜索与 EFI/UEFI 相关的信息
dmesg | grep -i "efi"

# 从内核日志 (dmesg) 中筛选出 UEFI 固件版本信息，确认系统是否以 UEFI 模式启动，并显示固件版本
dmesg | grep -i "efi: EFI v"

# 查看 UEFI 启动项信息（仅在 UEFI 启动模式下有效）
efibootmgr

# 查看 UEFI 固件的位数（32 位或 64 位）
cat /sys/firmware/efi/fw_platform_size

# 查看 Secure Boot（安全启动） 的当前状态
mokutil --sb-state
```
