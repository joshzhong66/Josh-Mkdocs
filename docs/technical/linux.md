# Linux

## 常用快捷键

```
##############################################################################
# 常用快捷键（默认使用 Emacs 键位）
##############################################################################

CTRL+B              # 向前移动
CTRL+F              # 向后移动
CTRL+A              # 移动到行首
CTRL+E              # 移动到行尾
CTRL+C              # 结束当前命令
CTRL+H              # 删除光标左边的字符
CTRL+K              # 删除光标位置到行末的内容
CTRL+L              # 清屏并重新显示
CTRL+N              # 移动到命令历史的下一行，同 <Down>
CTRL+O              # 类似回车，但是会显示下一行历史
CTRL+P              # 移动到命令历史的上一行，同 <Up>
CTRL+T              # 交换前后两个字符
CTRL+U              # 删除字符到行首
CTRL+W              # 删除光标左边的一个单词

ALT+数字			   # 切换窗口
```





ps命令

ps -p 16005 -o pid,cwd,cmd   	# 查看进程的 PID 和工作目录
